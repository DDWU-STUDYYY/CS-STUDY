(선택 절차 : 현재 상태에서 최적의 답을 찾는다. 적절성 검사 : 해가 문제의 조건을 만족하는지 찾는다. 해답 검사 : 문제가 해결이 되었으면 체크를 하고 아니면 다시 돌아간다)
