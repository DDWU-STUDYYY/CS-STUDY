## 방식 

1. 매주 월요일마다 이슈를 등록한다.
2. 이슈에 이번 주 목표를 작성한다.
3. 공부 내용을 노션이나 IDE에서 .md로 작성한다.
4. 공부한 내용을 [카테고리에 맞는 디렉토리/내 이름 디렉토리] 아래에 push 한다.
5. 투두메이트나 카카오톡으로 인증한다.
6. 파일 날짜 형식. ex)0217_KJH (날짜와 이름 이니셜)
